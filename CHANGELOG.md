# Changelog

All notable changes to this package will be documented in this file.

## 0.1.0 - 2026-06-14

### Added

- Added the initial browser-first TypeScript SDK.
- Added typed API groups for auth, recommendations, profiles, likes, ratings,
  prompts, connections, settings, chat, persistence, and raw requests.
- Added proxy transport contracts for browser-safe Hinge and Sendbird access.
- Added Sendbird realtime event parsing and proxy WebSocket support.
- Added npm build, typecheck, tests, docs, and examples.
